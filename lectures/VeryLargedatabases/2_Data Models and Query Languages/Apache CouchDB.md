An **open-source**, document-oriented [[NoSQL]] database that stores data as [[JSON]] documents, known for its reliability, distributed architecture and robust replication, making it excellent for offline-first apps, mobile and web applications needing high [[availability]] and [[scalability]] without rigid schemas. Built in [[Erlang]], it uses [[JavaScript]] for [[MapReduce queries]], offers a [[RESTful HTTP API]], and features automatic master-master replication for seamless data synchronization across devices and servers.
